Como Rodar os Testes

Os testes do projeto estão contidos na pasta "test", nela há um script python responsavel por realizar as comparações do resultado de saida do codigo com o resultado esperado. Para rodar os testes basta executar o comando python3 test.py, Caso queira adicionar um novo teste deve-se criar um novo arquivo input.txt (entrada para o codigo principal ), assim como um arquivo output.txt (resultado esperado), a estrutura destes arquivos deve seguir os padroes do projeto. representado pela figura a seguir.

Em seguida deve-se adicionar o nome destes arquivos recem criados a lista de testes que o scripit ira executar
